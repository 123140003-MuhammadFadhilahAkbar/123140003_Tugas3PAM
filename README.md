## 📖 Tentang Aplikasi

Aplikasi profil pribadi berbasis **Compose Multiplatform** yang menerapkan konsep UI deklaratif, reusable composable functions, layout dasar, modifier, dan animasi.

---

## ✨ Fitur Utama

| Fitur | Keterangan |
|-------|------------|
| 🖼️ Hero Header | Foto profil circular dengan gradient & status badge |
| 📊 Stats Card | Menampilkan Angkatan, Prodi, dan Jumlah Proyek |
| 📝 Bio | Deskripsi singkat tentang diri |
| 📞 Info Kontak | Email, telepon, lokasi, institusi |
| 🎞️ Animasi | `AnimatedVisibility` pada section kontak |

---

## 🧩 Composable Functions

```
HeroHeader   →  Header dengan foto profil dan gradient
StatsRow     →  Row berisi 3 StatCard
StatCard     →  Kartu statistik individual  
BioCard      →  Kartu bio/deskripsi diri
ContactCard  →  Kartu informasi kontak
InfoItem     →  Baris item kontak (emoji + label + value)
```

---

## 🛠️ Yang Digunakan

- **Layout** — `Column` `Row` `Box`
- **Komponen** — `Text` `Button` `Card` `Image` `Surface`
- **Modifier** — `padding` `clip` `background` `border` `fillMaxWidth`
- **Animasi** — `AnimatedVisibility` `expandVertically` `fadeIn` `fadeOut`
